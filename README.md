# linkup.salasa.id

![logo](/assets/icons/linkup-logo.png)

> Linkup is a simple contact book that allows you to manage your contacts easily.
> [linkup.salasa.id](https://linkup.salasa.id)

## Features

> There are two modes: `dark`, `light`

- Search contacts
- Contact list
  - Add Contact
  - Edit Contact
  - Delete Contact
- Trash
  - Restore Contact
  - Delete Contact
- Label
  - Add Label
  - Edit Label
  - Delete Label
